Alloc User Documenation

Alloc is a project/task allocation system for tracking people/projects/tasks.

Its main features are:

  • very simple to use
  • web based
  • password/account protected
  • allows projects to be created
  • allows phases to be allocated to projects
  • allows tasks to be allocated to phases
  • allows sub-tasks to be allocated to tasks
  • allows email reminders to be sent to staff who have tasks to be done
  • allows for timeline (pseudo-Gantt-style) charts to be displayed onscreen
  • will automatically determine projected completion dates, based on inputs
