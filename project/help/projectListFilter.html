Help - Project List Filter

The project list filter allows you to limit the list to projects that match the criteria that you enter. The filter is shown above the project list. The following criteria may be available, depending on your permissions:

Status Limits the list to projects allocated to those set to the selected status (potential, current or archived). The status is set using the Project Edit page.
Type Limits the list to projects allocated to those set to the selected type (project, job or contract). The status is set using the Project Edit page.
Allocated to Limits the list to projects allocated to the seleted person. People are allocated to projects by managers using the Project Edit page.
Name containing Limits the list to projects with names that contain the entered text.

Once you have entered your criteria, click the Apply button to load the list using your criteria.
