Help - Client List Filter

The client list filter allows you to reduce the number of clients listed by specifying some criteria. You can specify the following criteria:

Status Limits the list to clients allocated to those set to the selected status (potential, current or archived). The status is set using the Client Edit page.
Name containing Limits the list to clients with names that contain the entered text.

Once you have entered your criteria, click the Apply button to load the list with your criteria applied.
