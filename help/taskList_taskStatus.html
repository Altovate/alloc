Task Status

Incomplete: Tasks that don't have an Actual Completion Date.
Started: Incomplete Tasks that have an Actual Start Date.
New Tasks: Incomplete Tasks that have been created in the last 2 working days.
Due Today: Incomplete Tasks that have a Target Completion Date = today.
Overdue: Incomplete Tasks that have a Target Completion Date prior to today.
Completed: Tasks that have an Actual Completion Date.
