Saved Task Filters

This feature allows you to save specific filters for future use. To load a filter, select the filter name from the dropdown list and then click Load.

To save a filter, select the filter settings you would like to save. Then, click the Saved Filters link to show the additional filtering controls, enter a descriptive name for the filter into the Name: box and then click Save. You can also delete saved filters by selecting the filter name from the list and clicking Delete.
