Saved Filters

This feature allows you to save and re-use specific filter settings.

To save a new filter:

  • Adjust the Task Filter settings below as desired.
  • Select "Create a new saved filter..." from the Saved Filter dropdown.
  • Enter a descriptive name for this Filter setting.
  • Click the "Save" button

To load an existing filter, just select the filter from the dropdown list and click the "Load" button

To delete a filter, first select the unfortunate Filter from the dropdown list, and then click the "Delete" button.
