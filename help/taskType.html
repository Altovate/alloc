Task
Regular Tasks use this Task Type.

Parent/Phase
When you want a Task that you can add subtasks to, use this Task Type.

Message
If the priority for this task is 1 then a reminder for this task will be created and sent off every day to the person this task is assigned to, until the task is closed. If the priority is not 1 then the email will get sent every X days where X is the tasks priority.

Fault
If the Priority for this task is 1 then a reminder will be sent to ALL the members of this tasks project every four hours until this task is closed. If the Priority for this task is 2 then a reminder will be sent to this tasks assignee and the project manager until the task is closed.

Milestone
This task type is used to nominate milestones. These tasks appear on task graphs slightly differently.
