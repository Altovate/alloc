Save
The Save button will save the Billing Note, and whether you are electing to use Payment Insurance. It does not save the time sheet line items below, and it will not take you away from this page.

Delete
This button will erase the current time sheet from the system. All the line entries that you have entered will be deleted.

To Manager
You will only see this button here if a manager has been selected for this particular project. This button will save the current time sheet and submit it to the manager. You will not be able to edit this time sheet any further if you click this button, unless the manager rejects it and decides to make the time sheet editable again for some reason.

To Admin
This button will save the current time sheet and submit it to the time sheet administrator. You will not be able to edit this time sheet any further if you click this button, unless the admin rejects it and either sends it back to the manager or simply decides to make it editable again.
