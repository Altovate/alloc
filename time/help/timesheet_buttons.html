Delete Record

This button will erase the current timesheet from the system. You will lose all the line entries that you have entered.


Save

The Save button will save the info above, which project the timesheet is for, whether you are electing to use Payment Insurance. It does not save the time sheet line items below, and it will not take you away from this page.


Re-Grab Pay Details

This button will grab the most recent payment details from the project area and plug them into this timesheet. Typically you only need to click this button if your payment details have changed on the project page and you need to refresh this timesheet.


Save and Submit to Manager

You will only see this button here if a manger has been selected for this particular project. This button will save the current timesheetand submit it to the manager. You will not be able to edit this timesheet any further if you click this button, unless the manager rejects it and decides to make the timesheet editable again.


Save and Submit to Admin

This button will save the current timesheetand submit it to admin. You will not be able to edit this timesheet any further if you click this button, unless the admin rejects it and either sends it back to the manager or simply makes it editable again.
